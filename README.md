# Salary Prediction System

An end-to-end machine learning project that predicts salary based on user demographics, education, experience, and job role.

## Features
- Data preprocessing and cleaning
- Missing value imputation
- Categorical encoding
- Random Forest Regression
- Streamlit web app deployment

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## Run Locally
```bash
streamlit run app.py
